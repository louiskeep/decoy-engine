## Developer Certificate of Origin (DCO)

We use the [Developer Certificate of Origin](https://developercertificate.org/) — a simple per-commit attestation that you have the right to submit your contribution. There is no separate Contributor License Agreement to sign.

To sign off, add a `Signed-off-by:` trailer to each commit by passing `-s` to `git commit`:

```bash
git commit -s -m "Your commit message"
```

This adds a line like:

```
Signed-off-by: Your Name <your.email@example.com>
```

The `Name` and `email` must match your real name and a contactable email address. Pseudonymous contributions are not accepted.

A DCO-check GitHub Action is planned; until it lands, please sign off manually.
